# Disclaimer

AI has been used to assist the development of this software.

## How AI was used in this project:
The entire codebase for AvatarLockpick was written with the assistance of Artificial Intelligence, specifically using Google Antigravity and the Windsurf Editor.

Just to be clear this project is roughly 50% Human and 50% AI:

- The Human (Me): The core idea, the HTML GUI, the frontend layout and the foundational app backend were all conceptualized, designed and hand-coded by me.

- The AI: The AI tools were used to assist the entire codebase by editing, optimizing and improving my hand-made code, as well as providing a few specific functional additions.

## Note
I understand that there is a lot of skepticism and distaste surrounding the use of AI in communities right now. Those concerns are entirely valid. However, the reality of modern-day development is that AI has become an incredibly powerful utility.

Using AI does not replace a human developer, rather it acts as a highly capable pair-programmer, like an assitsant or like a stack overflow. For a project like AvatarLockpick, utilizing AI allowed me to speed up development time, troubleshoot bugs and refine the code far more efficiently than I could have done entirely on my own. It handles the heavy lifting for a lot of things and optimization so I can focus on making a user-friendly, functional tool for people.

Every line of code generated or edited by AI was prompted, reviewed and finalized by a human (me). The vision, the logic and the quality control remain 100% mine.

Thank you for understanding and thank you for supporting me. 💖
