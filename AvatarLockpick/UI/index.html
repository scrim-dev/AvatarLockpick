How it works

Connects to your Quest or Android device via ADB (Android Debug Bridge) over USB. ADB pulls avatar cache files from /sdcard/Android/data/com.vrchat.oculus.quest/files/LocalAvatarData/, runs the database unlock on each file, then pushes them back. Enable Developer Mode / USB Debugging on your headset before connecting.

Step 1: Preparation

Launch VRChat and switch into the avatar you wish to unlock. Wait for the avatar to load completely, then enter your VRChat User ID and the Avatar's ID into the application.

Step 2: Unlocking

Attempt to unlock the avatar using the standard Unlock method. If the avatar uses VRCFury locks, use Unlock (VRCFury) or Database Unlock instead.

Step 3: Restart

Restart your game using the restart buttons within the app. Enjoy your newly unlocked avatar!
